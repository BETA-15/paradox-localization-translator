# パラドゲー ローカライズ一括翻訳ツール

Paradox Interactive系ゲーム（CK3・Victoria3・HoI4・Stellaris・EU5等）全般対応のMod/実績ローカライズを、ローカルLLM（Ollama）でまとめて日本語化するツールです。

## できること
- `localization/english`（または`simp_chinese`）フォルダをそのまま突っ込むだけで自動翻訳
- ゲーム内変数・カラーコード（`[GetName]` `$VALUE$` `§Y...§!` など）を検出して保護、翻訳後も壊さず復元
- 英語・簡体字中国語のどちらからでも和訳可能（自動判定）
- 複数行をまとめて1リクエストで処理するバッチ方式なので、キー数が多いファイルでも比較的高速
- 翻訳結果はキャッシュされるので、Mod更新時は差分だけ再翻訳・中断しても再開可能
- 対応ゲームを限定していないので、Paradoxタイトルであれば大抵のMod・実績・イベント文に使えます

## 必要なもの
- Mac（Windows版コマンドは未整備、中身のPythonスクリプトは動くはず）
- [Ollama](https://ollama.com) をインストールして起動しておくこと
- 適当な日本語対応モデル（`qwen3.6:latest`など）をpullしておくこと

## 使い方
1. ZIPをダウンロード・解凍
2. `原文`フォルダの中に`english`（or `simp_chinese`）フォルダごとMod内のlocalizationデータを入れる
3. `翻訳する.command`をダブルクリック
4. `翻訳後`フォルダに`*_l_japanese.yml`が出てくるので、Modのlocalizationフォルダに配置

詳細な使い方・トラブルシューティングは同梱の`使い方.md`を参照してください。
